Cette API permet de déterminer à l'aide d'un triplet (nom de commune, année de naissance, département de naissance) le code INSEE de la commune de naissance.

Exemple de commande (tous les paramètres sont obligatoires): 

curl https://identite-pivot-code-insee-naissance-lookup.fly.dev \
    -X GET \
    -d 'nom_commune=Gennevilliers' \
    -d 'annee_naissance=1960' \
    -d 'departement_commune=92'
